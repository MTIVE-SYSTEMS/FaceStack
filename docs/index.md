# FaceStack documentation

Fast, accurate **face recognition** (1:N identification) with an optional **body
(person ReID)** mode that recognises someone when their face is not visible —
usable as a Python library or a REST/WebSocket service.

> **For AI agents:** the full integration spec is inlined on this page below.
> For the cleanest machine-readable copies, fetch any of:
> - **Raw Markdown (best for LLMs):** `https://raw.githubusercontent.com/MTIVE-SYSTEMS/FaceStack/main/docs/integration.md`
